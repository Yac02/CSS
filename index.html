Style CSS

Maneras para vincular CSS con HTML

  1. Enlazando archivo CSS
    • Se debe crear un archivo "Style.css" y enlazarlo en el index con un link>
    • EJEMPLO: < link rel="stylesheet" href="styles.css" >
  2. Atributo Style
    • Se puede utilizar directamente en el index seguido de la etiqueta
    • EJEMPLO: < p style="font-size: 18px">Este es un párrafo con estilos inline.< / p>
  3. Etiqueta < Style >
    • Se puede utilizar directamente en el HTML como una etiqueta que engloba el contenido
    • EJEMPLO: < head> < style > h1 { color : green; < / style > < / head >

Reglas de CSS

  1. Selector
    • Es como el nombre de algo a lo que quieres aplicar estilo. Puede ser el nombre de una etiqueta HTML (h1, p, div), una clase (.clase) o un ID (#identificador). Así se invoca al elemento que se desea poner estilo.
  2. Propiedad
    • Son las características que quieres cambiar o ajustar en los elementos seleccionados. Por ejemplo, el color del texto (color), el tamaño de la letra (font-size).
  3. Valor
    • Es lo que se le da a la propiedad para decirle cómo se debe ver. Por ejemplo en el caso del color, #00ff00 (un código de color).
